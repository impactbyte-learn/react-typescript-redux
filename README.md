# React Typescript w/ Redux & React Router

# Includes

- React & ReactDOM
- React Router
- Redux
- Redux Logger (development)
- Redux Thunk
- Parcel Bundler
- TSLint (linter)
- Jest (unit testing)
- Enzyme
- Service Workers (PWA)
- Dockerfile
- etc.

## How to use

```bash
npm install

npm run dev # Development server

npm run build # Production build
npm run start # Production server

npm run test # run all tests
```
